## 🚀 简介

本项目旨在集成主流开源大模型，实现不同类型大模型以及同类型大模型之间的协调合作，从而提供全面而灵活的AI生成内容服务。

## 📚 文档

[文档](https://doc.aigc.wenliang.space)

## 🛠️ 技术栈

### 后端

Java SpringBoot3 MySQL

### 前端

Vue3

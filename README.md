# 项目介绍

## 简介
本项目旨在集成主流开源大模型，实现不同类型大模型以及同类型大模型之间的协调合作，从而提供全面而灵活的AI生成内容服务。

## 功能
### 文本转语音
平台利用多种先进的AI模型，包括GPT-SoVITS、Fish-Speech、ChatTTS和Edge-TTS，为小说中的对话和角色生成高质量的语音。
用户可以上传文本内容，在操作界面进行章节拆分，并为每个角色生成相应的语音配音。

#### 使用的技术
1. GPT-SoVITS
2. Fish-Speech
3. ChatTTS
4. Edge-TTS
